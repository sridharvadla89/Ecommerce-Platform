# Ecommerce-Platform
Full-stack eCommerce web application built using React and Spring Boot with product search, category filtering, cart functionality, and REST APIs.
  Features

-  Product Search (Live search with API)
-  Add to Cart functionality
-  Category Filtering
-  Dark Mode Toggle
-  Product Listing from Backend
-  REST API Integration

---

🛠️ Tech Stack

Frontend
- React.js
- Bootstrap
- Axios

Backend
- Spring Boot
- Spring Data JPA
- REST APIs

 Database
- H2
